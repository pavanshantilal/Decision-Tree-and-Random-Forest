Decision Tree and Random Forest Classification

AI & ML Internship - Task 5

Objective

The objective of this project is to understand and implement Decision Tree and Random Forest algorithms for classification using the Heart Disease Dataset.

Dataset

* Heart Disease Dataset (heart.csv)

Tools Used

* Python
* Pandas
* Matplotlib
* Scikit-learn

Algorithms Used

* Decision Tree Classifier
* Random Forest Classifier

Project Workflow

1. Load the dataset
2. Check dataset information
3. Split data into training and testing sets
4. Train a Decision Tree Classifier
5. Visualize the Decision Tree
6. Analyze overfitting using different tree depths
7. Train a Random Forest Classifier
8. Compare model accuracies
9. Display feature importance
10. Evaluate the model using 5-Fold Cross Validation

Results

* Decision Tree Accuracy
* Random Forest Accuracy
* Cross Validation Accuracy
* Feature Importance Graph
* Decision Tree Visualization
* Overfitting Analysis Graph
